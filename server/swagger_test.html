Common Issues:

  1. Server not running: python manage.py runserver
  2. Static files not collected: python manage.py collectstatic
  3. drf-spectacular not installed: pip install drf-spectacular
  4. Browser cache: Clear cache and refresh

Debug Steps:

  1. Check server console for errors
  2. Open browser developer tools (F12)
  3. Check Network tab for failed requests
  4. Check Console tab for JavaScript errors
